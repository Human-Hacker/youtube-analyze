## 0. 分析方針

1. **原因と結果を分離**: 総再生数・IMP数は「結果」。CTR・視聴深度・エンゲージメント率が「原因」
2. **人間評価GI×CAスコア使用**: AI自己評価(17-24の狭い範囲)は不採用。人間評価(5-23)を使用
3. **対数スケール相関**: 再生数はべき乗分布 → log₁₀(再生数)で相関を取る
4. **3段階フィルター**: アルゴリズムの評価プロセスを再現

## 9. 核心的インサイト

### 第一原理から導かれる因果関係
```
アーティスト選定(GI) × 切り口(CA) → 初動CTR(F1) → Day2成長(F2)
    → 視聴深度×エンゲージメント(F3) → アルゴリズム増幅 → 総再生数
```

### 重要な発見
1. **経過時間が最大の交絡変数**: 古い動画ほど再生数蓄積 → VPD正規化が必須
2. **GI×CAは足切りとして機能**: 全動画がGI×CA≧15で素材品質は十分。差は他の要因
3. **VPD正規化後、文字数のみが真の正の相関**: 台本の情報密度が日あたり再生数に寄与
4. **MV挿入数・メディア数はVPD正規化後に消失**: 経過時間との交絡（見せかけの相関）
5. **GI_v3（素材の質）はVPD正規化後も弱い相関**: アーティスト選定は一定の影響あり

### アクション
- **制作前**: GI×CA >= 40 のアーティスト×切り口を推奨（全体の底上げ済み）
- **台本**: 文字数（情報密度）を重視。4要素完備 + フック詐欺回避
- **制作**: MV2箇所以上挿入（直接的効果は不明だが視聴体験に寄与）
- **分析**: 総再生数ではなくVPD（日あたり再生数）で動画パフォーマンスを評価
