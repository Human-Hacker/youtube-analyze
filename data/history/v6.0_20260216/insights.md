# 分析インサイト

## 採択済みインサイト

### 因果モデル

```
アーティスト選定(GI) × 切り口(CA) → 初動CTR(F1) → Day2成長(F2)
    → 視聴深度×エンゲージメント(F3) → アルゴリズム増幅 → 総再生数
```

制作前の選定（GI×CA）が唯一の有効な予測因子。制作後指標（CTR、年齢層、性別比、MV挿入数等）はGI×CAが高ければほぼ影響しない。

### 1. GI×CAモデル（信頼度: 高）
- **GI (Gossip Index) v3**: G1(ゴシップ)+G2(好奇心の強さ)+G3(感情)+G4(映画化)+G6(楽曲知名度)。各1-5、合計5-25
- **CA (Curiosity Alignment)**: 動画の切り口が日本人の好奇心TOP1-2にどれだけ答えているか。0-3
- **GI×CA ≥ 16**: 成功閾値。r=0.856, R²=0.732, 精度92.9%（14本中13本正解）
- GI単体 vs log(再生数): r=0.795 → GIだけでも有用だがCAを掛けると精度向上
- CA単体 vs log(再生数): r=0.671 → CA=0は致命的
- データ: 14本の人間評価済み。10本は未評価（要人間レビュー）
- **注意**: R²=0.732は「約73%の分散を説明」であり「80-90%を決定」は過大表現。14本・2変数のin-sample精度であり、out-of-sample検証は未実施
- **灰色ゾーン**: GI×CA=16-30では予測不確実性が高い（アリアナGI×CA=24→MISS vs ビリーGI×CA=22→HIT）

### 2. 原因と結果の分離（信頼度: 高 → 初回サイクルで強化）
- **原因指標**（制作側がコントロール可能）: GI×CA（アーティスト選定×切り口）のみが有効な予測因子
- **結果指標**（伸びた「結果」として発生）: 総再生数, IMP数, 登録者視聴数, 新規視聴者率, ブラウジングIMP量, SUBSCRIBER比率, RELATED比率, 年齢層分布
- 結果指標と再生数の相関（r=0.9+）は因果ではなく共変動。予測因子として使ってはいけない
- **初回サイクルでの検証**: 12本の仮説（H5-H16）のうち大半が結果指標を予測因子として使用し棄却された。この原則の重要性が再確認された

### 3. Broadcast Dilution Effect（信頼度: 高）
- エンゲージメント率 vs 再生数: r=-0.697（強い負の相関）
- 原因: アルゴリズムが大量配信すると、受動的な視聴者比率が増え、いいね/コメント率が下がる
- 含意: 低エンゲージメント率 = 悪い動画ではない。むしろ大成功の証拠

### 4. 登録者パラドックス（信頼度: 高）
- 公開時登録者数 vs 再生数: r=-0.612（負の相関）
- 因果が逆: ヒット動画がチャンネルを育てた→後の動画ほど登録者は多いが伸びるとは限らない
- 含意: 登録者の多さは「過去のヒットの遺産」であり、未来の成功を保証しない

### 5. CTRは足切りのみ（信頼度: 高）
- ブラウジングCTR vs log(再生数): r=+0.038（ほぼ無相関）
- CTR ≥ 4.0% は必要条件だが十分条件ではない
- 高CTRでも中身が期待外れならDay2で暴落する（フック詐欺パターン）
- カレンカーペンター(CTR=10.04%)、マドンナ(CTR=9.4%)は高CTRだがMISS → GI×CAが低ければCTRが高くても伸びない

### 6. GI×CAの頑健性（信頼度: 高 — 初回サイクルで新規確立）
- **ポールマッカートニーの全パターン破壊**: age55+=83.9%(最高), MV=0, 男性比率73.7%(最高) という全ての「不利な指標」を持ちながらHIT（202,214再生, GI×CA=45）
- GI×CA ≥ 16の動画に対して、制作後指標（年齢層、性別比、MV挿入数）はほぼ影響しない
- 制作後指標が良好でもGI×CAが低ければMISS（カレンカーペンター、マドンナ）
- **結論**: 「誰を選ぶか×どの切り口か」が支配的。制作技術は従属的

### 7. 3段階フィルターモデル（信頼度: 低 — 初回サイクルで引き下げ）
- F1: ブラウジングCTR ≥ 4.0%
- F2: Day1→Day2変化率 > -20%
- F3: 平均視聴≥300秒 AND エンゲージメント率≥0.8%
- 現在の精度: 66.7%（24本中16本正解）→ ほぼランダムに近い
- F3にBroadcast Dilution Effectの裏返しが混入（HIT動画のeng率が下がるため偽陰性が出る）
- **閾値の根本的見直しが必要。現時点では実用的な予測ツールではない**

### 8. 台本構造パターン（信頼度: 低 — 初回サイクルで引き下げ）
- **4要素完備**: ratio=0.69（完備群の方が平均再生数が低い逆転が発生）→ 効果は確認できない
- **MV挿入**: MV=0でもHIT（ポールマッカートニー202K）→ GI×CAとの交絡の可能性
- **フック回答あり**: フック詐欺回避の意義は残るが、定量的効果は不明
- **感情エスカレーション**: 単独での効果は確認できない

### 9. AI GIスコアの系統的過大評価（信頼度: 高 — 初回サイクルで新規確立）
- AI生成GIは人間評価より系統的に高い（過大評価バイアス）
- ビリーアイリッシュ: AI=21 vs 人間=11（差+10）
- アリアナグランデ: AI=24 vs 人間=12（差+12）
- ジャスティンビーバー: AI=23 vs 人間=19（差+4）
- 人間GIが低いほどAIとの乖離が大きい（AIは下限を過大評価する）
- **AI生成GIスコアは予測に使用禁止。人間評価のみを使用すること**

- 精度66.7%はほぼランダム → 閾値の根本的見直しまたはモデル廃止を検討
