# ケースコントロール研究法（Case-Control Methodology）

> このファイルは Step 6（ペア選定エージェント）専用のドメイン知識である。
> N=24の小標本で有意な知見を得るための方法論を提供する。

---

## 0. 第一原理: YouTube動画が伸びる公式

```
Views = Impressions × CTR × f(Retention)
```

- **Impressions**: アルゴリズムが動画を表示した回数（制作者はコントロール不可）
- **CTR**: クリック率。サムネイル×タイトル×トピック×タイミングで決まる
- **f(Retention)**: 維持率がアルゴリズムに与える正のフィードバック。維持率が高い→推薦が増える→Impressionsが増える

つまり: **CTRで人を呼び、Retentionでアルゴリズムを味方につける**。この2つが揃わなければ伸びない。

---

## 1. ケースコントロール研究法とは

### 1.1 基本概念
- **ケース（Case）**: 注目する結果（HIT = 15万回以上）を持つ動画
- **コントロール（Control）**: 結果が異なる（MISS = 15万回未満）動画
- **マッチング**: ケースとコントロールの「交絡因子」を揃えることで、真の因果因子を特定する

### 1.2 なぜペア比較なのか（N=24の制約）
- N=24では統計的な相関分析（回帰分析、t検定等）の信頼性が低い
- 統計的有意性を追求するより、**条件統制された比較**の方が有用
- 「似た条件で結果が違うペア」を比較することで、因果要因を絞り込める
- 全体の傾向分析（Phase 2-3）では見えない個別の因果関係を発見できる
- ペア比較は「差分」に焦点を当てるため、交絡因子の影響を最小化できる

### 1.3 ペア比較の論理
```
ペアA = {HIT, GI=24, ロック, 2月公開}
ペアB = {MISS, GI=22, ロック, 2月公開}

GI・ジャンル・時期が近似 → これらは結果差の原因ではない
→ 残る差分（タイトル設計、台本構造等）が因果候補
```

---

## 2. N=24での交絡因子制御

### 2.1 交絡因子とは
- 原因と結果の両方に影響を与える第三の変数
- 例: 「アーティストの知名度が高い→再生数が多い」場合、知名度は交絡因子
- マッチングで交絡因子を揃えれば、残る差分が因果候補になる

### 2.2 マッチングで制御すべき因子
| 因子 | 制御方法 | 制御しないリスク |
|------|---------|-----------------|
| Topic Power (GI) | GI差3以内でマッチ | 知名度が高いから伸びた、という自明な結論になる |
| 音楽ジャンル | 同ジャンルでマッチ | ジャンルの視聴者層の違いが交絡 |
| 公開時期 | 1ヶ月以内でマッチ | チャンネル成長フェーズの違いが交絡 |
| 動画長 | 3分以内の差でマッチ | 動画長そのものの影響が交絡 |

### 2.3 N=24特有の制約
- 24本中、HIT/MISSの分布が偏っている可能性がある
- マッチング条件を厳しくしすぎるとペアが見つからない
- マッチング条件を緩めすぎると交絡因子の制御が不十分
- **推奨**: 4条件のうち2つ以上が合致するペアを選定

---

## 3. マッチング基準設計

### 3.1 マッチング条件（以下の1つ以上が近似するペア）
1. **GIスコアが近い**: GI差が3以内（例: GI=24 vs GI=22）
2. **同ジャンル**: 同じ音楽ジャンル（ロック、ポップ、R&B等）
3. **同時期公開**: 公開日が1ヶ月以内
4. **動画長が近い**: 長さの差が3分以内

### 3.2 結果の差
- 一方がHIT（15万回以上）、他方がMISS（15万回未満）
- または、再生数に2倍以上の差がある

### 3.3 除外条件
- 遅延HIT（Day7以降に伸びた動画）は外部要因の可能性が高いため、別枠で扱う
- 両方HITまたは両方MISSのペアは対象外

### 3.4 品質スコアリング
各ペアに「マッチング品質スコア」を付与:

| 条件 | スコア |
|------|--------|
| GI差1 | 3点 |
| GI差2 | 2点 |
| GI差3 | 1点 |
| 同ジャンル | 2点 |
| 同時期公開（1ヶ月以内） | 1点 |
| 動画長差3分以内 | 1点 |

**最大 7点**。品質スコアの高い順に3〜5ペアを選定。

### 3.5 カバレッジの考慮
- 同じ動画が複数ペアに含まれすぎないようにする
- HIT群・MISS群の多様性を確保する（偏った動画のみにならないよう注意）
- 最低3ペア、最大5ペアを選定

---

## 4. HIT/MISS閾値の根拠

### 4.1 閾値: 15万回再生
- ウタヨミチャンネル（登録者約47,000人）において、15万回は登録者の約3倍
- この閾値はアルゴリズム推薦による拡散が発生したことを示す
- 15万回未満はブラウジング推薦の初期拡大が不十分だったことを意味する

### 4.2 チャンネル概要（参考）
- **コンテンツ**: 海外音楽アーティストの伝記・物語形式の長尺動画（13〜22分）
- **動画数**: 24本（2025年7月〜2026年2月）
- **登録者推移**: 約20,000人 → 約47,000人（約136%成長）
- **公開頻度**: 週1〜2本

---

## 5. 指標分類（原因 vs 結果）

### 5.1 原因指標（制作者がコントロール可能 = ペア差分の注目対象）
- アーティスト選定（Topic Power）
- タイトル・サムネイル設計（Hook Power）
- 台本構造: フック設計、MV配置、感情曲線、導入30秒（Retention Power）
- 公開タイミング（曜日、時事性）

### 5.2 結果指標（伸びた結果として発生 = ペア差分の確認用）
- 再生数、IMP数、ブラウジングIMP量、登録者視聴数
- 新規視聴者率、SUBSCRIBER比率、RELATED比率
- 年齢層分布、性別比率
- Day1視聴数、Day1→Day2変化率
- エンゲージメント率、平均視聴時間、平均視聴率

**ペア選定では**: 結果指標はペアの「結果差の確認」に使用する。原因指標は「注目すべき差分候補」として列挙する。
