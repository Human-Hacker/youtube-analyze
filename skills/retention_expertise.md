# 維持率分析の専門知識（Retention Expertise）

> このファイルは Step 4（維持率分析エージェント）専用のドメイン知識である。
> 維持率曲線と台本構造の因果関係を解明するための理論・手法を提供する。

---

## 0. 第一原理: YouTube動画が伸びる公式

```
Views = Impressions × CTR × f(Retention)
```

- **Impressions**: アルゴリズムが動画を表示した回数（制作者はコントロール不可）
- **CTR**: クリック率。サムネイル×タイトル×トピック×タイミングで決まる
- **f(Retention)**: 維持率がアルゴリズムに与える正のフィードバック。維持率が高い→推薦が増える→Impressionsが増える

つまり: **CTRで人を呼び、Retentionでアルゴリズムを味方につける**。この2つが揃わなければ伸びない。

**維持率分析エージェントとして**: この公式の右辺 f(Retention) が担当領域。
維持率が高ければアルゴリズム推薦が増え、Impressionsが雪だるま式に増加する。
台本構造のどの要素がf(Retention)を押し上げるかを特定することが使命。

## 1. ブラウジング推薦と維持率のフィードバック

- YouTube のホームフィードに表示されるかどうかは、主に**CTR（クリック率）× 視聴維持率**で決まる
- 新規動画はまず登録者のフィードに表示され、初動のCTRと視聴維持が良ければ非登録者にも拡大される
- Day1のブラウジング推薦の量がその後の伸びを大きく左右する
- **維持率は推薦拡大の「エンジン」**: 初期の維持率が良い→推薦される→新規視聴者が流入→さらに維持率が高ければ推薦が加速

---

## 2. 感情移入と物語構造（維持率の心理的メカニズム）

### 2.1 感情同一化による視聴継続
- 視聴者は主人公（アーティスト）の感情に同一化することで視聴を継続する
- 「逆境→努力→成功」「栄光→転落→再起」等の物語構造は感情移入を促進する
- 感情の「底」（最も辛い場面）が深いほど、その後の回復に対するカタルシスが大きい

### 2.2 楽曲の力（音楽チャンネル固有）
- 知名度の高い楽曲がMVとして挿入されると、感情のピークを人工的に作れる
- 「聴きたかった曲」が物語のクライマックスで流れる体験は視聴満足度を大きく左右する
- 楽曲を知らない視聴者にとってはMV挿入が離脱ポイントになり得る

### 2.3 維持率に影響する台本要素（分析対象）
| 台本要素 | 維持率への影響メカニズム |
|---------|----------------------|
| 導入30秒 | 第一印象で「見続ける価値があるか」を判断。離脱の最大ポイント |
| フック設計 | 「この問いの答えを知りたい」で視聴を継続させる |
| 感情曲線 | 感情の起伏（UP/DOWN/転換）が視聴者の注意を維持する |
| 底の深さ | 底が深いほどカタルシスへの期待で離脱しにくい |
| MV挿入 | 知名曲は感情ピーク、無名曲は離脱リスク |
| 敵役 | 敵役がいると物語に緊張感が生まれ、視聴が継続する |
| エスカレーション | 後半に向けて事態が深刻化すると離脱しにくい |

## 3. ウタヨミ視聴維持パターン（24本の実績データ）

- **平均視聴時間**: 5〜8分（動画長の29〜39%）
- **関連動画からの視聴者**は登録者より長く視聴する傾向（414〜582秒 vs 299〜464秒）
- **含意**: 新規視聴者ほど「期待を持ってクリック→満足すれば長く見る」パターン
- **動画長**: 13〜22分の長尺動画。維持率30%台は業界標準的だが、改善余地あり

## 4. 指標分類（原因 vs 結果）

### 4.1 原因指標（制作者がコントロール可能 = 分析対象）
- 台本構造: フック設計、MV配置、感情曲線、導入30秒（Retention Power）
- アーティスト選定（Topic Power — 維持率にも間接的に影響）
- 公開タイミング（曜日、時事性）

### 4.2 結果指標（伸びた結果として発生 = 分析対象外）
- 再生数、IMP数、ブラウジングIMP量、登録者視聴数
- 新規視聴者率、SUBSCRIBER比率、RELATED比率
- 年齢層分布、性別比率
- Day1視聴数、Day1→Day2変化率
- エンゲージメント率、平均視聴時間、平均視聴率

**重要**: 平均視聴時間・平均視聴率は「結果指標」である。
維持率分析では、これらを「観測値」として使うが、「原因」として使ってはならない。
「高い維持率がHITを生む」のではなく「良い台本構造が高い維持率を生む」が正しい因果方向。
