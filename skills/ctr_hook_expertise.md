# CTR・フック分析の専門知識（CTR & Hook Expertise）

> このファイルは Step 5（CTR分析エージェント）専用のドメイン知識である。
> クリック率を決定する要因と、GI/CAスコアリング基準を提供する。

---

## 0. 第一原理: YouTube動画が伸びる公式

```
Views = Impressions × CTR × f(Retention)
```

- **Impressions**: アルゴリズムが動画を表示した回数（制作者はコントロール不可）
- **CTR**: クリック率。サムネイル×タイトル×トピック×タイミングで決まる
- **f(Retention)**: 維持率がアルゴリズムに与える正のフィードバック。維持率が高い→推薦が増える→Impressionsが増える

つまり: **CTRで人を呼び、Retentionでアルゴリズムを味方につける**。この2つが揃わなければ伸びない。

**CTR分析エージェントとして**: この公式の CTR が担当領域。
CTRは「見るかどうかの瞬間」を決める指標であり、サムネイル×タイトル×トピック×タイミングの関数。
何がクリックを生むかのパターンを解明することが使命。

---

## 1. YouTubeアルゴリズムとCTR

### 1.1 ブラウジング推薦（ホームフィード）
- YouTube のホームフィードに表示されるかどうかは、主に**CTR（クリック率）× 視聴維持率**で決まる
- 新規動画はまず登録者のフィードに表示され、初動のCTRと視聴維持が良ければ非登録者にも拡大される
- Day1のブラウジング推薦の量がその後の伸びを大きく左右する
- サムネイルとタイトルがCTRを決定し、コンテンツの質が視聴維持率を決定する

### 1.2 関連動画推薦
- 現在視聴中の動画と「類似性が高い」と判断された動画が関連動画として表示される
- 類似性はトピック、視聴者の行動パターン（同じ人が見ている動画）、メタデータで判定される
- 関連動画からの流入は、特定のトピック（アーティスト名）への検索需要と連動する

### 1.3 YouTube検索
- 検索トラフィックは視聴者の能動的な意図を反映する
- 時事性のあるイベント（訃報、受賞、スキャンダル等）で検索需要が急増する
- 検索トラフィックの急増は遅延HITの主要トリガーの一つ

---

## 2. クリックを生む視聴者心理

### 2.1 好奇心ギャップ（Curiosity Gap）
- 人は「知りたいこと」と「現在知っていること」のギャップがあると、それを埋めようとする欲求が生じる
- タイトル・サムネイルでギャップを作り、動画内で段階的に解消することで視聴維持が上がる
- ギャップが大きすぎると「釣り」と判断されて離脱、小さすぎると「見なくてよい」と判断される

### 2.2 ゴシップ欲求
- 有名人のスキャンダル、裏話、知られていない事実への関心は普遍的に強い
- 「みんなが知っているあの人の、みんなが知らない話」が最もクリック率が高い
- ゴシップ要素が強すぎるとネガティブな反応を招くリスクもある

---

## 3. ウタヨミのトラフィック構成

- **主要流入**: 登録者フィード（ブラウジング）38〜74% + 関連動画 22〜50%
- **この2つで全体の70〜96%**を占める。検索・外部は合計5%未満
- **含意**: アルゴリズム推薦への依存度が極めて高い
- ブラウジングCTRが「入口」、関連動画CTRが「横展開」を決定する

## 4. GI/CAスコアリング定量基準

> 以下は `scoring_criteria.md` v1.0（2026-02-16）の全文転記。
> GIサブスコア(G1-G6)とCAスコアの主観的判断を排除し、全指標を定量的に計測可能にする。

### 4.1 GIサブスコア定量基準

#### G1: ゴシップ露出度（Web検索カウント）

**検索クエリ**: `"{アーティスト名}" ワイドショー OR 特集 OR スキャンダル`
**対象期間**: 直近2年（故人の場合: 死去年を含む3年間 + 直近2年の合計）
**計測値**: 独立記事数（同一ニュースの転載は1件としてカウント）

| スコア | 独立記事数 |
|--------|-----------|
| 5      | 10件以上  |
| 4      | 5-9件     |
| 3      | 3-4件     |
| 2      | 1-2件     |
| 1      | 0件       |

#### G2: 好奇心の総量（Googleサジェスト分析）

**検索方法**: `"{アーティスト名}"`をGoogle検索窓に入力した際のサジェスト補完を確認
**計測値**: サジェスト中の「好奇心系ワード」の出現数

**好奇心系ワード一覧**: なぜ, 理由, 死因, 現在, 真実, 真相, 秘密, 衝撃, 闇, 何があった, どうなった

| スコア | 好奇心系ワード数 |
|--------|-----------------|
| 5      | 5個以上         |
| 4      | 3-4個           |
| 3      | 2個             |
| 2      | 1個             |
| 1      | 0個             |

#### G3: 感情的フックの強さ（イベントカウント方式）

**計測方法**: research.md/data_summary.md から以下のカテゴリ該当数をカウント
**判定基準**: 事実として確認できるもののみカウント（噂・未確認情報は除外）

**カテゴリ一覧**:
| ID | カテゴリ | 例 |
|----|---------|-----|
| A  | 若年死亡（50歳未満で死去） | アヴィーチー(28歳), カレン(32歳) |
| B  | 重病/障害（不治の病、身体障害、精神疾患） | ガガ(線維筋痛症), エド(吃音・難聴) |
| C  | 虐待/暴力被害（DV、性暴行、いじめ） | ガガ(19歳性暴行), ビリー(いじめ) |
| D  | 依存症（薬物、アルコール、ギャンブル） | エルトン(コカイン), ブルーノ(ギャンブル) |
| E  | 貧困/ホームレス経験 | エド(路上生活), ブルーノ(廃墟育ち) |
| F  | 家族崩壊（親の虐待、毒親、離婚による苦難） | カレン(母), MJ(父) |
| G  | 自殺未遂/自殺 | アヴィーチー(自殺), エルトン(未遂) |
| H  | 権力者による搾取（マネージャー、レーベル等） | テイラー(マスター音源), ブリトニー(後見制度) |
| I  | 事件/テロ/暗殺の被害者 | アリアナ(テロ), JL(暗殺) |
| J  | 復活/逆転（底からの這い上がり） | JB(ディディ後の復活), エルトン(依存症克服) |

| スコア | 該当カテゴリ数 |
|--------|---------------|
| 5      | 6個以上       |
| 4      | 4-5個         |
| 3      | 3個           |
| 2      | 2個           |
| 1      | 0-1個         |

#### G4: 映画/ドキュメンタリー認知（日本公開実績）

**検索クエリ**: `"{アーティスト名}" 映画 日本 公開 OR 興行収入`
**計測値**: 日本で最もヒットした伝記映画/ドキュメンタリーの興行規模

| スコア | 基準 |
|--------|------|
| 5      | 日本興行収入10億円以上の伝記映画 |
| 4      | 日本興行収入1-9億円の伝記映画 |
| 3      | 日本で劇場公開されたドキュメンタリー（興行1億未満） |
| 2      | 配信のみ（Netflix/Disney+等）で日本語対応あり |
| 1      | 伝記作品なし、または日本未公開 |

**注意**: フィクション映画（アリー/スター誕生等）は対象外。伝記映画・ドキュメンタリーのみ対象

#### G6: 国民的楽曲知名度（CM/カラオケ/TV実績カウント）

**検索クエリ**: `"{アーティスト名}" CM OR カラオケ OR 紅白 OR ドラマ主題歌`
**計測値**: 日本のCM/TV番組/カラオケで使用実績が確認できる楽曲数

| スコア | 実績確認楽曲数 |
|--------|---------------|
| 5      | 3曲以上       |
| 4      | 2曲           |
| 3      | 1曲           |
| 2      | CM/TV使用実績なし。カラオケランキング上位のみ |
| 1      | 上記いずれの実績も確認できない |

### 4.2 好奇心マップ（TOP1-2）定量取得方法

#### 必須検索クエリ（5本）

| # | クエリ | 目的 |
|---|--------|------|
| 1 | `{アーティスト名} なぜ` | 疑問・不思議系の関心 |
| 2 | `{アーティスト名} 死因` or `{アーティスト名} 現在` | 生死/近況への関心 |
| 3 | `{アーティスト名} 日本` | 日本接点 |
| 4 | `{アーティスト名} 衝撃` or `{アーティスト名} 真実` | スキャンダル・暴露系 |
| 5 | `{アーティスト名}`（単体） | Googleサジェスト・関連検索の確認 |

#### TOP1-2の決定方法

1. 5検索の結果から、繰り返し登場するトピックを抽出
2. 各トピックの「出現クエリ数」（5本中何本で出現したか）を記録
3. 出現クエリ数が多い順にTOP1-5をランキング
4. TOP1-2を確定し、各TOPの**核心キーワード**を2-3語で抽出

**核心キーワード抽出ルール**:
- TOP文から名詞・動詞を抽出（助詞・形容詞は除外）
- 最も具体的な語を選ぶ（例: 「病気」より「線維筋痛症」）
- 2-3語に限定

### 4.3 CAスコア定量基準

#### キーワード照合方式

1. TOP1-2の核心キーワードを用意（上記で抽出済み）
2. 動画タイトル全文と照合

| CA値 | 照合基準 |
|------|---------|
| 3    | タイトルにTOP1の核心キーワードが**1語以上**直接含まれる |
| 2.5  | タイトルにTOP2の核心キーワードが**1語以上**直接含まれる（TOP1は不一致） |
| 2    | タイトルにTOP1-2の**関連語**（上位語・類義語）が含まれる |
| 1    | タイトルにTOP1-2のキーワード・関連語なし。TOP3以下のキーワードのみ |
| 0    | タイトルのテーマが好奇心マップ（TOP1-5）に存在しない |

### 4.4 エビデンス記録要件

全スコアについて、以下を`human_scores.json`に記録すること:

| フィールド | 内容 |
|-----------|------|
| `source` | `"quantitative"` |
| `G{N}_evidence` | 検索クエリ、計測結果、判定根拠 |
| `CA_evidence` | TOP1-2核心KW、タイトル照合結果 |
| `curiosity_top1` | TOP1テキスト（Web検索結果由来） |
| `curiosity_top2` | TOP2テキスト（Web検索結果由来） |
| `curiosity_search_date` | 検索実施日 |

## 5. 指標分類（原因 vs 結果）

### 5.1 原因指標（制作者がコントロール可能 = 分析対象）
- アーティスト選定（Topic Power）
- タイトル・サムネイル設計（Hook Power）
- 台本構造: フック設計、MV配置、感情曲線、導入30秒（Retention Power）
- 公開タイミング（曜日、時事性）

### 5.2 結果指標（伸びた結果として発生 = 分析対象外）
- 再生数、IMP数、ブラウジングIMP量、登録者視聴数
- 新規視聴者率、SUBSCRIBER比率、RELATED比率
- 年齢層分布、性別比率
- Day1視聴数、Day1→Day2変化率
- エンゲージメント率、平均視聴時間、平均視聴率

**重要**: B-CTRは「結果指標」であるが、Topic Power（GI）× Hook Power（CA）との相関を分析する際の「観測値」として使用する。CTRそのものを予測因子にしてはならない。
