# 4軸フレームワーク分析の専門知識（Four-Axis Analysis）

> このファイルは Step 7（ケーススタディ分析エージェント）専用のドメイン知識である。
> HIT/MISSペアの差分を4軸で深掘り分析するための完全なフレームワークを提供する。

---

## 0. 第一原理: YouTube動画が伸びる公式

```
Views = Impressions × CTR × f(Retention)
```

- **Impressions**: アルゴリズムが動画を表示した回数（制作者はコントロール不可）
- **CTR**: クリック率。サムネイル×タイトル×トピック×タイミングで決まる
- **f(Retention)**: 維持率がアルゴリズムに与える正のフィードバック。維持率が高い→推薦が増える→Impressionsが増える

つまり: **CTRで人を呼び、Retentionでアルゴリズムを味方につける**。この2つが揃わなければ伸びない。

## 1. 4軸フレームワーク完全定義

### 1.1 Topic Power（「この人の話を聞きたい」力）
アーティスト固有の興味・関心度。制作者はアーティスト選定でコントロール。

| 指標 | 説明 |
|------|------|
| GI (General Interest) | G1〜G6の加重平均。アーティストの総合的関心度 |
| G1 (ゴシップ露出度) | ワイドショー・スキャンダル記事の量 |
| G2 (好奇心の総量) | Googleサジェストの好奇心系ワード数 |
| G3 (感情的フックの強さ) | 人生の劇的イベント数（若年死亡、虐待、依存症等） |
| G4 (映画/ドキュメンタリー認知) | 日本での伝記映画の興行規模 |
| G6 (国民的楽曲知名度) | CM/カラオケ/TV使用実績 |
| 時事性 | アーティスト関連ニュースとの連動 |
| 公開タイミング | 需要のピークとの合致 |

### 1.2 Hook Power（「このサムネ/タイトルをクリックしたい」力）
タイトル・サムネイルが視聴者のクリック欲求をどれだけ刺激するか。

| 指標 | 説明 |
|------|------|
| CA (Curiosity Alignment) | タイトルキーワードとTOP1-2好奇心の一致度 |
| B-CTR | ブラウジングCTR（観測値） |
| タイトルパターン | 疑問型/対比型/事実提示型/エモーショナル型 |
| TOP1一致 | タイトルが好奇心TOP1のキーワードを含むか |

### 1.3 Retention Power（「最後まで見たい」力）
台本構造が視聴者を引き留める力。

| 指標 | 説明 |
|------|------|
| 平均視聴時間 | 実測値（結果指標だが観測用） |
| 平均視聴率 | 動画長に対する視聴割合（結果指標だが観測用） |
| 導入30秒タイプ | 衝撃事実型/問いかけ型/エピソード導入型 |
| 引きの強さ | 導入30秒のフック強度（1-5） |
| 感情転換数 | UP/DOWN/転換の合計 |
| 底の深さ・数 | 感情の「底」のカウント |
| MV挿入数 | 楽曲MV挿入箇所の数 |
| フック回答位置 | オープニング質問への回答タイミング |

### 1.4 Virality Power（「誰かに教えたい」力）
動画が自発的に拡散される力。

| 指標 | 説明 |
|------|------|
| エンゲージメント率 | (いいね+コメント+シェア)/再生数 |
| コメント率 | コメント数/再生数 |
| シェア数 | 共有された回数 |
| 登録者獲得数 | 動画経由の新規登録者 |

## 2. 軸間相互作用理論

### 2.1 4軸の独立性と連鎖
- 各軸は独立した因果経路を持つが、**順序的な連鎖**がある
- Topic Power → Hook Power → Retention Power → Virality Power
- Topic Power が高くても Hook Power が低ければCTRが低い
- Hook Power が高くても Retention Power が低ければ推薦されない
- 「どの軸が不足してMISSになったか」を動画ごとに特定することが分析の核心

### 2.2 軸の組み合わせパターン
| パターン | 特徴 | 結果 |
|---------|------|------|
| 全軸高 | 理想型 | 確実にHIT |
| Topic+Hook高、Retention低 | 「フック詐欺型」 | CTR高いが推薦が伸びない |
| Topic高、Hook低 | 「隠れた良作型」 | 潜在力は高いがクリックされない |
| Retention高、Topic低 | 「ニッチ良作型」 | 見た人は満足だがリーチが限定的 |
| 全軸低 | 「失敗型」 | 確実にMISS |

### 2.3 因果の方向
```
Topic Power → CTR（一部）→ 初動再生 → Retention判定 → 推薦拡大 → Virality発生
              Hook Power ↗
```
- Topic Power と Hook Power は「入口」を決める
- Retention Power は「定着」を決める
- Virality Power は「拡散」を決める（最も外部要因に依存）

## 3. 差分分析の因果推論手法

### 3.1 差分分析の基本手順
1. ペアの両動画の4軸データを並べる
2. 各軸の差分の大きさを定量化する
3. 最も大きな差分を特定する
4. その差分が「原因指標」かどうか確認する
5. 原因指標である場合、因果仮説として記録する

### 3.2 因果候補の判定基準
- **制作者がコントロール可能**であること（原因指標）
- **HIT側が一貫して優位**であること（ペア横断で確認）
- **メカニズムが説明可能**であること（なぜその因子がHITに寄与するか）

### 3.3 ペア横断パターン分析
- 全ペアの差分を横断し、共通する因果パターンがあるか確認する
- 「全ペアでHIT側が優れている軸」= 強い因果候補
- 「一部ペアでのみHIT側が優れている軸」= 条件付き因果候補
- 因果候補のランキングを作成する

## 4. YouTubeアルゴリズムの仕組み（差分解釈の背景知識）

### 4.1 ブラウジング推薦（ホームフィード）
- YouTube のホームフィードに表示されるかどうかは、主に**CTR × 視聴維持率**で決まる
- 新規動画はまず登録者のフィードに表示され、初動のCTRと視聴維持が良ければ非登録者にも拡大される
- Day1のブラウジング推薦の量がその後の伸びを大きく左右する

### 4.2 関連動画推薦
- 現在視聴中の動画と「類似性が高い」と判断された動画が関連動画として表示される
- 類似性はトピック、視聴者の行動パターン、メタデータで判定される

### 4.3 YouTube検索
- 検索トラフィックは視聴者の能動的な意図を反映する
- 時事性のあるイベントで検索需要が急増する

### 4.4 成長パターン
- **即時成長型**: Day1-2で大部分の視聴が発生（ブラウジング推薦が強い）
- **持続成長型**: Day1-7で安定的に成長（関連動画+検索が継続）
- **遅延成長型**: Day7以降に急伸（外部イベント or バイラル発生）
- **減衰型**: Day1をピークに急落（フック詐欺 or 需要ミスマッチ）

## 5. 視聴者心理の原則（差分解釈の心理学的根拠）

### 5.1 好奇心ギャップ
- 「知りたいこと」と「現在知っていること」のギャップが視聴動機
- タイトル・サムネイルでギャップを作り、動画内で段階的に解消

### 5.2 感情移入と物語構造
- 視聴者は主人公の感情に同一化することで視聴を継続する
- 感情の「底」が深いほど、その後の回復に対するカタルシスが大きい

### 5.3 ゴシップ欲求
- 有名人の裏話、知られていない事実への関心は普遍的に強い
- 「みんなが知っているあの人の、みんなが知らない話」が最もクリック率が高い

### 5.4 楽曲の力
- 知名度の高い楽曲のMV挿入は感情のピークを作れる
- 楽曲を知らない視聴者にとってはMV挿入が離脱ポイントになり得る

## 6. チャンネルベンチマーク（ウタヨミ固有データ）

### 6.1 チャンネル概要
- **コンテンツ**: 海外音楽アーティストの伝記・物語形式の長尺動画（13〜22分）
- **動画数**: 24本（2025年7月〜2026年2月）
- **登録者推移**: 約20,000人 → 約47,000人（約136%成長）

### 6.2 視聴者プロファイル
- **コア視聴者層**: 45〜64歳の女性（全視聴の30〜40%）
- **サブ視聴者層**: 35〜44歳の男女、65歳以上の女性
- **含意**: 懐かしさ（ノスタルジア）への共感が視聴動機の中核

### 6.3 トラフィック構成
- **主要流入**: ブラウジング 38〜74% + 関連動画 22〜50%
- **合計**: 70〜96%がアルゴリズム推薦由来
- **含意**: アルゴリズム推薦への依存度が極めて高い

### 6.4 維持パターン
- **平均視聴時間**: 5〜8分（動画長の29〜39%）
- **関連動画視聴者**: 登録者より長く視聴する傾向

### 6.5 HIT/MISS閾値
- **HIT**: 15万回以上（登録者の約3倍。アルゴリズム推薦拡大の証）
- **MISS**: 15万回未満

## 7. 指標分類（原因 vs 結果）

### 7.1 原因指標（制作者がコントロール可能 = 差分分析の核心）
- アーティスト選定（Topic Power）
- タイトル・サムネイル設計（Hook Power）
- 台本構造: フック設計、MV配置、感情曲線、導入30秒（Retention Power）
- 公開タイミング（曜日、時事性）

### 7.2 結果指標（伸びた結果として発生 = 差分の確認用）
- 再生数、IMP数、ブラウジングIMP量、登録者視聴数
- 新規視聴者率、SUBSCRIBER比率、RELATED比率
- 年齢層分布、性別比率
- Day1視聴数、Day1→Day2変化率
- エンゲージメント率、平均視聴時間、平均視聴率

### 7.3 GI/CAスコアの定義（参照用）

**GIサブスコア**:
| サブスコア | 内容 | 計測方法 |
|-----------|------|---------|
| G1 | ゴシップ露出度 | Web検索での独立記事数 |
| G2 | 好奇心の総量 | Googleサジェストの好奇心系ワード数 |
| G3 | 感情的フックの強さ | 劇的人生イベントのカテゴリ数 |
| G4 | 映画/ドキュメンタリー認知 | 日本での伝記映画の興行規模 |
| G6 | 国民的楽曲知名度 | CM/カラオケ/TV使用実績の楽曲数 |

**CAスコア**:
| CA値 | 基準 |
|------|------|
| 3 | タイトルにTOP1の核心キーワードが直接含まれる |
| 2.5 | タイトルにTOP2の核心キーワードが直接含まれる |
| 2 | タイトルにTOP1-2の関連語が含まれる |
| 1 | TOP3以下のキーワードのみ |
| 0 | 好奇心マップに存在しない |
