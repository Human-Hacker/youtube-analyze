# YouTube動画分析エージェント

YouTube長編動画の成功要因を第一原理から解明し、再現可能な予測モデルを構築する13ステップAIパイプライン。

## ファイル構成

| フォルダ | 役割 |
|---------|------|
| `agents/` | 各ステップのプロンプト（Phase別サブフォルダ） |
| `skills/` | ドメイン知識ファイル（エージェント専用。1 Agent = 1 Skill） |
| `commands/` | パイプライン実行コマンド |
| `scripts/` | Python実行スクリプト |
| `data/` | 入出力データ |

詳細: `workflow_files.md`

## 依存の原則（厳守）

1. **一方向依存**: 各ステップは直前Phaseの出力のみ参照。2つ以上前に遡らない
2. **final出力の情報内包**: 各Phaseのfinal出力は後続Phaseが必要とする全情報を含む（付録テーブル）
3. **1 Agent = 1 Skill**: 各エージェントは専用skillを1つだけ参照する。共有skillは存在しない
4. **処理済みファイル共有禁止**: エージェント間で処理済み出力を共有しない（checker例外のみ許可）
5. **ドメインパック方式**: Phase 1はエージェント別に専用データパックを生成。data_summary.mdはスクリプト専用
6. **1 Phase = 1 Purpose**: 各Phaseは単一の目的を持つ。複数目的の混在は禁止

## 4軸モデル

分析の中核となる4次元モデル:

| 軸 | 意味 | 主な指標 |
|----|------|----------|
| Topic Power | 「この人の話を聞きたい」力 | GI, Google Trends, 競合密度 |
| Hook Power | 「このサムネ/タイトルをクリックしたい」力 | CA, CTR, タイトル設計 |
| Retention Power | 「最後まで見たい」力 | 維持率曲線, 台本構造, MV配置 |
| Virality Power | 「誰かに教えたい」力 | コメント率, シェア率, エンゲージメント |

## 命名規則

`step{N}_*.py` = スクリプト / `step{N}_*.md` = エージェントプロンプト

## 出力ルール

- 日本語で出力
- markdownファイルとしてdata/output/に保存
- 出力パスは `commands/analyze-pipeline.md` の各ステップ定義に従う
